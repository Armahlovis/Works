Deep Learning

Introduction

  1. Deep Learning is a learning approach that enable computers to solve human level problems using hierracy of concepts

  2. Hierracy of concept allow computers to learn complicated concept by building them out of simpler one

  3. Knowledge used by computers can be hard-coded in form of knowledge base or acquired by ML algorithms

  4. Machine learning algorithm is affected by the representation of the input data. In represenation learning ML learns the represenation itself. Eg. Autoencoders

  5. Depth of a model is measured by computational graph or depth of proability modeling graph

Historical Trends

  1. Deep Learning, in the past, has been referred to by names such as cybernetics, connectionism and Aritificial Neural Network

  2. Increased amount of training data has made deep learning more useful

  3. Models have increased in size and complexity due to improved infrastructure

  4. Deep learning has solved increasingly complicated applications with increasing accuracy.
